# CryptoPayMap

CryptoPayMap is a verified discovery service for physical places and online services where customers can pay from a crypto wallet.

The project is currently in its initial specification and development setup phase. Public product, data, verification, and implementation documentation will be added through reviewed pull requests.

## Core principle

Published listings are intended to identify the payment asset, network, route, instructions, evidence, and last confirmation date.

## Status

Early development. The public repository structure and specifications are being established.
